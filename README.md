# rust-template
Rust template 

Install Rust

1. Install

```curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh```

Or use the Bash script on Terminal 

```./install_rust.sh```

2. Source

```source "$HOME/.cargo/env"```

Or use Makefile command 

```make source```

3. Check version

```which rustc```
